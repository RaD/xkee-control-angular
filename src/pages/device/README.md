# Компонент "Устройство"

Роутер `areas/:pk/devices/create` и `areas/:pk/device/:device_pk/:action`.

## Функциональность

Форма для создания и редактирования устройства с полями:
* идентификатор (UUID, только при создании)
* название (обязательное)
* описание (обязательное, минимум 3 символа)

## Действия

* назад - возврат к списку устройств
* применить - сохранение изменений
* удалить - переход к подтверждению удаления

## Особенности

- Условное отображение поля идентификатора (только при создании)
- Валидация полей с визуальными индикаторами
- Автоматическое удаление при action='delete'
- Адаптивный дизайн для всех устройств
- Интеграция с сервисом хранения данных
- Автоматический редирект после операций
